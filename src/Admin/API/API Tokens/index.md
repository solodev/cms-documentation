# API Tokens

An API token is similar to a password and allows you to authenticate to Dataverse Software APIs to perform actions as you. Many Dataverse Software APIs require the use of an API token.

On the API tokens page, you have the list of existing tokens. Each token is shown with details such as the token name, status, and a token secret that can be previewed using the eye icon.
