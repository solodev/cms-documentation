# Creating a Package

This article shows users how to create a package to deploy to any Solodev installation.  

!!! Note: 
Before building packages, users should have a finalized design and a clear sitemap. To learn more, please review the Building a custom website section in the Solodev documentation. 
!!!

## Creating a package

* Create a website. 

<img src="/static/images/create-a-package.png" alt="document-overview" style="width: 100%; display: block"></a>

* After creating the website, open the website and click **Settings**.

<img src="/static/images/create-settings.png" alt="document-overview" style="width: 100%; display: block"></a>

* In the modal window click the **Advanced** tab to access the advanced options.

<img src="/static/images/create-advanced.png" alt="document-overview" style="width: 50%; display: block"></a>

* Click **Export Package**. 

<img src="/static/images/create-export-package.png" alt="document-overview" style="width: 50%; display: block"></a>

* The user is directed to a modal window that provides options to customize the export. 
* Click the checkbox to **Include manager entries in export**. 
* Click **Continue Export**.

<img src="/static/images/create-continue-export.png" alt="document-overview" style="width: 50%; display: block"></a>

* The user is directed to modal window to download the package. 
* Click **Download Package**. 

<img src="/static/images/create-download.png" alt="document-overview" style="width: 50%; display: block"></a>

* The zip file will be found in the folder that is assigned to Downloads in the web browser.   

<img src="/static/images/crerating_packages_7.jpg" alt="document-overview" style="width: 100%; display: block"></a>
