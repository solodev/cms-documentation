# Redirects

URL/page redirects can help direct users away from pages that no longer exist or to more preferred locations. To do that, you need to create an .htaccess file with the creation of each website.

><img src="../../../images/redirects4.jpg" alt="redirects4" style="width: 70%; display: block"></a>

## Actions

**Name** | **Description** 
:--- | ---
History | Click "History" button to see the history of changes in relation to redirects.

## Fields

**Name** | **Description** 
:--- | ---
Code | Enter code snippet to create redirect.

---

### History

Here you can check history of changes in relation to redirects.

><img src="../../../images/redirects5.jpg" alt="redirects5" style="width: 70%; display: block"></a>

#### Actions

**Name** | **Description** 
:--- | ---
Download version | Click to download selected version of the .htaccess file.
Compare | Click to compare previous version of the .htaccess file with current file.
Save as current version | Save the previous version as current version.

#### Fields

**Name** | **Description** 
:--- | ---
Code | Here you can edit code snippet.

---

## Confirm

Click the blue "Submit" button at the bottom right to apply your changes.
