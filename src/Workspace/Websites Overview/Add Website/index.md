# Add Website

## Website Address

Enter website address.

## Package

Upload package (optional).

><img src="../../../images/websites-add3.jpg" alt="websites-add3.jpg" style="width: 20%; display: block"></a>

## Website Timezone

Choose website timezone

><img src="../../../images/websites-add4.jpg" alt="websites-add4.jpg" style="width: 60%; display: block"></a>

## Description

Enter description.

><img src="../../../images/websites-add5.jpg" alt="websites-add5.jpg" style="width: 60%; display: block"></a>

## Add

Once you’ve completed all of the fields, click the blue “Add” button and your website will be created.
